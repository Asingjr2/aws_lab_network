# AWS_LAB_NETWORK

### Purpose
- Project deploys full aws network using cloud resources

### Requirements
- aws account
- aws cli

### Implementation
- Access aws console and utilize templates in following order:
    - network.yaml
    - permissions.yaml
    - autoscaling.yaml

- route tables
- subnets
